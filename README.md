# 🚀 Estructuras de Datos I - UNSAdA (2026)

Este repositorio contiene las implementaciones de estructuras de datos fundamentales desarrolladas durante el tercer año de la carrera de **Analista de Sistemas**. El objetivo es dominar la lógica de almacenamiento, eficiencia algorítmica y gestión de memoria.

## 🛠️ Tecnologías y Conceptos
* **Lenguaje:** Java 21
* **Entorno:** Visual Studio Code
* **Enfoque:** Implementación de TADs (Tipos de Datos Abstractos) y análisis de complejidad $O(n)$.

---

## 📂 Contenido del Repositorio

### 1. Pilas (Stacks) 🥞
Implementación de estructuras LIFO (*Last In, First Out*).
* **Implementación Básica**: Métodos `push`, `pop`, `top`, `empty` y `size` usando arreglos.
* **Validación de Paréntesis**: Algoritmo para verificar el balanceo de símbolos `() [] {}`.
* **Inversión de Secuencias**: Algoritmo para revertir el orden de $N$ elementos.

### 2. Colas (Queues) 👥
Estructuras FIFO (*First In, First Out*).
* [Pendiente] Implementación de colas simples y circulares.

### 3. Listas Enlazadas (Linked Lists) 🔗
Gestión dinámica de memoria mediante nodos.
* [Pendiente] Listas simplemente enlazadas, doblemente enlazadas y circulares.

### 4. Árboles (Trees) 🌳
Estructuras jerárquicas y no lineales.
* [Pendiente] Árboles Binarios de Búsqueda (BST) y recorridos (Inorder, Preorder, Postorder).

### 5. Hashing (Tablas Hash) 🔑
Optimización de búsqueda mediante funciones de dispersión.
* [Pendiente] Resolución de colisiones y tablas de direccionamiento abierto.

## 🔧 Cómo ejecutar los proyectos
1. Clonar el repositorio:
